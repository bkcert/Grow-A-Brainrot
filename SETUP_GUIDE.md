# Setup Guide for Grow-A-Brainrot

This guide provides step-by-step instructions on how to copy each script file into Roblox Studio for the project "Grow-A-Brainrot". Please follow the paths provided to ensure that each script is placed in the correct location.

## Step 1: Open Roblox Studio

1. Launch Roblox Studio.

## Step 2: Create a New Game or Open an Existing One

1. If you are starting a new project, select "Create New" and choose a template. 
2. If you already have an existing game, click on "My Projects" to open it.

## Step 3: Copy Script Files to the Correct Locations

### 1. Copy `MainScript.lua`
- **File Path:** `Scripts/MainScript.lua`
- **Destination in Roblox Studio:** 
  - Right-click on `ServerScriptService` in the Explorer panel, then select "Insert Object" and choose "Script". 
  - Paste the content of `MainScript.lua` into the newly created script.

### 2. Copy `PlayerHandler.lua`
- **File Path:** `Scripts/PlayerHandler.lua`
- **Destination in Roblox Studio:** 
  - Right-click on `ServerScriptService` in the Explorer panel, then select "Insert Object" and choose "Script". 
  - Paste the content of `PlayerHandler.lua` into the newly created script.

### 3. Copy `UIHandler.lua`
- **File Path:** `Scripts/UIHandler.lua`
- **Destination in Roblox Studio:** 
  - Right-click on `StarterGui` in the Explorer panel, then select "Insert Object" and choose "LocalScript". 
  - Paste the content of `UIHandler.lua` into the newly created local script.

### 4. Copy `DataStoreHandler.lua`
- **File Path:** `Scripts/DataStoreHandler.lua`
- **Destination in Roblox Studio:** 
  - Right-click on `ServerScriptService` in the Explorer panel, then select "Insert Object" and choose "ModuleScript". 
  - Paste the content of `DataStoreHandler.lua` into the newly created module script.

## Step 4: Save Your Game

1. After copying all scripts, click on "File" from the menu and select "Save".

## Conclusion

You have successfully copied the necessary script files into Roblox Studio. Make sure to test your game to ensure everything is functioning as expected!
